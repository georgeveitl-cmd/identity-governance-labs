# Identity Governance Labs

This repository contains educational lab documentation demonstrating identity governance and access management concepts used in enterprise environments.

The focus of these labs is to demonstrate how organizations manage identity lifecycle, privileged access, and role-based access control (RBAC) in modern cloud and hybrid environments.

## Topics Covered

• Role-Based Access Control (RBAC) design  
• Access review processes  
• Privileged access monitoring  
• Service account governance  
• Identity lifecycle management  

## Purpose

These labs demonstrate governance concepts commonly used in Identity and Access Management (IAM) and Identity Governance & Administration (IGA) programs.

All examples are hypothetical and do not contain any proprietary or production environment information.
